WebShield

A rule-based phishing detection system that analyzes URLs using multiple security indicators and classifies them as Safe, Suspicious, or Phishing.

URL Input

The user enters a URL into the system for analysis. The rule-based engine evaluates the link using multiple phishing indicators and assigns a risk score to classify the URL as Safe, Suspicious, or Phishing.

Problem Statement

Phishing attacks are one of the most common cybersecurity threats where attackers create fake websites to steal user credentials and financial information. Many users cannot easily distinguish between legitimate and malicious links. This project aims to develop a lightweight rule-based system to identify suspicious URLs.

  • PhishTank – Community-based phishing database.

Research Gap

Most existing phishing detection systems rely on complex machine learning models. This project focuses on a lightweight, explainable rule-based approach suitable for real-time detection.
