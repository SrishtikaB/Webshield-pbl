WebShield

This project implements a rule-based phishing detection system that analyzes URLs using multiple security indicators. It assigns a score based on suspicious features and classifies links as Safe, Suspicious, or Phishing.

Problem Statement

Many users fall victim to phishing and malicious links. It is difficult for common users to identify unsafe URLs. This project aims to check the safety of a URL using predefined rules.

Methodology

Input

User enters a URL for analysis.

Rule-Based Analysis

The system checks URL length, HTTPS usage, special characters, suspicious keywords, IP-based URLs, and domain extensions to calculate a phishing score.

Results
